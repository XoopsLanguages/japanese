ヘルプ：基本設定

説明

基本設定パネルでは、サイト全体の設定を構成できます。設定項目の中には、パフォーマンス、セキュリティ、サイトの使いやすさに大きな影響を与えるものもあるため、時間をかけて確認・設定することをおすすめします。

一般設定

最も重要で頻繁に利用される設定項目です。ここでは、サイト名やスローガンを設定します。また、デフォルトのテーマや管理画面インターフェース（Admin GUI）を選択できます。メンテナンス時には、サイト全体を一時的に非公開にすることも可能です。モジュールごとのキャッシュ時間の設定も行えます。

メタタグとフッター

メタタグは、あなたのウェブサイトの内容を検索エンジンに伝えるための情報です。主に検索エンジンやクローラーがサイトを正しくインデックス化するために使われます。設定したメタ情報は、すべてのページのヘッダーに埋め込まれ、現在はページやモジュールごとに個別設定することはできません。

検索エンジンによってインデックス化の方法は異なり、メタタグに重みを置くものもあれば、無視するものもあります。メタ情報を適切に設定することで検索順位を改善できる可能性があります。アクセスログなどを分析し、どの検索エンジンが主な訪問元かを確認して、その方針に合わせて調整するのが効果的です。

メタ情報は正確かつ誠実に記述しましょう。検索エンジンを「操作」して順位を上げようとする行為は、検出されるとサイトが評価を下げられたり、インデックスから除外される可能性があります。

メタ著作権とフッターフィールドでは、メンテナンスを簡素化するために以下の置換タグが使用できます：
 - {X_SITEURL}：サイトのトップページURLに置き換えられます。
 - {X_YEAR}：現在の4桁の西暦年に置き換えられます（毎年の更新不要）。

ワード検閲オプション

XOOPS には、投稿された不適切な言葉を自動で置換する機能があります。置換語は任意の単語で指定可能です。この機能はシステムリソースを消費するため、サイトの速度に若干影響する可能性があります。デフォルトでは無効です。

検閲を有効にするには「有効」のラジオボタンを選択します。検閲したい単語は「検閲ワード」欄にパイプ（|）区切りで入力します（大文字小文字は区別されません）。「置換後の語」欄には適切な置換語を入力します。初期値は「#OOPS#」です。

検索オプション

検索機能は、ポータル型サイトにおいて情報を探すための非常に重要なツールです。XOOPS の検索機能は、インストールされているモジュールに応じてニュース、ダウンロード、リンクなどのカテゴリに分類して結果を表示します。

サイト全体のグローバル検索はデフォルトで有効です。これを無効にすると、検索機能は使用できなくなり、検索ブロックも非機能になります。
検索時のキーワードの最小文字数は通常5文字です。これより短い単語は無視されます。短い単語（例：「サイトマップ」など）は曖昧な結果が多くなる傾向があり、精度の低下を招きます。長いキーワードは結果件数が減る反面、精度が高くなります。

検索処理はサーバーに負荷がかかるため、短い単語を除外することでパフォーマンス向上に寄与します。ただし、短い語を検索したいケースもあるので、必要に応じて調整しましょう。

メール設定

このセクションでは、XOOPSシステムのメール配信設定を行います。デフォルト設定の「PHP (mail)」のままにしておくことを推奨します。別の送信方法（Sendmail や SMTP）を使用しない限り、それらのフィールドを記入する必要はありません。

認証オプション

通常は「Xoops データベース」のままで構いません。他の設定は無視して構いません。

LDAP や Microsoft Active Directory（ADS）サーバーによる認証がサポートされています。これは企業などで既存のユーザーデータベースを活用してログインさせるための機能です。ユーザーは新たにXOOPS用アカウントを登録する必要がなくなります。

システムモジュール設定

このセクションでは、各システム機能の有効/無効を切り替えたり、アバターや画像の表示件数などのシステム全体のデフォルト設定を行うことができます。また、ブロックやコメント、各モジュールで使用するデフォルトエディタの選択も可能です。
