XOOPS のインストール

初回インストール

はじめに：

XOOPS は、PHP で記述されたオープンソースのオブジェクト指向 Web パブリッシングシステムです。小規模から大規模な動的なコミュニティ Web サイト、社内ポータル、企業ポータル、Web ログなどの開発に最適なツールです。

XOOPS は、 GNU General Public License (GPL) バージョン 2 以降の条項に基づいてリリースされており、自由に使用および変更できます。 GPL の配布条項を遵守する限り、自由に再配布できます。

必要条件

  • WWW サーバー (Apache, NGINX, IIS など)
  • PHP 5.6.0 以上、7.4 以上を推奨
  • MySQL 5.6 以上、MySQL 8.0 以上を推奨

インストールする前に

  1. WWW サーバー、PHP、データベースサーバーを適切にセットアップします。
  2. XOOPS サイト用のデータベースを準備します。
  3. ユーザーアカウントを準備し、ユーザーにデータベースへのアクセスを許可します。
  4. ディレクトリとファイルを書き込み可能にします: %s
  5. セキュリティ上の理由から、以下の 2 つのディレクトリを ドキュメントルート の外に移動し、フォルダ名を変更することを強くお勧めします: %s
  6. (まだ存在しない場合は) ディレクトリを作成し、書き込み可能にします: %s
  7. ブラウザの Cookie と JavaScript をオンにします。

ローカルへのインストール

開発またはテストのためにローカル環境を実行している場合は、上記の 要件が満たされていることを確認してください。完了したら、htdocs ディレクトリ (XOOPS 配布ファイルから) の内容を Web 環境のルートドキュメントパスにコピーします。ファイルがコピーされたら、 ブラウザで "http://yoursite.com" を指定してインストールを開始できます。これにより、インストールプロセスが開始されます。

ホストされたプラットフォームへのインストール

ホスト環境で実行している場合は、XOOPS ファイルをローカルまたはサーバー (シェル (つまり SSH) アクセスがある場合) に展開します。完了したら、htdocs ディレクトリからルート Web ディレクトリ (プロバイダーは通常、この場所を指示します) にすべての XOOPS ファイルを移動またはコピーしてください。 ファイルがコピーされたら、ブラウザで "http://yoursite.com" を指定してインストールを開始できます。これにより、インストールプロセスが開始されます。
